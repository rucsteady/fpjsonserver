### Setup the Focus Positive App

Start a Terminal in each of these three Diretories

`focuspositive`
`fpjsonserver`
`chatServer`

install all dependencies in each dir with

### `npm install`

Start Service in each dir

`focuspositive`

### with `npm run dev`

`fpjsonserver`
`chatServer`

### with `npm start`
